*Instalacion de JME2

-Ir al home y adentro de workspace hacer:

svn co -r HEAD http://jmonkeyengine.googlecode.com/svn/trunk jme

-Una vez que bajo el src de JME, ir a ~/workspace/jme y correr el siguiente script:

sh lib/mvn-lib-install.sh

-Correr el build de JME2:

mvn clean install

-En eclipse, poner nuevo proyecto (tipo: Project) y en el nombre poner jme (el mismo que usamos para hacer checkout antes)...

-Bajar el proyecto de cg_final, habilitar maven, hacer update de las dependencias, de la configuracion del proyecto y abrir la configuracion del build path.

-Elegir Add Library, User library, crear una nueva con nombre JME y agregarle los siguientes jar que estan dentro del workspace/jme/lib/

lwjgl/jinput.jar
lwjgl/lwjgl.jar
lwjgl/lwjgl-util-applet.jar
jogl/jogl.jar
jorbis/jorbis.jar
junit/junit-4.1.jar

-Aceptar y editar el jar de lwjgl.jar, en Native library location poner jme/lib/lwjgl/native/linux

-Con esto deberia arrancar todo bien el ejemplo que deje...
