**HowTo para correr el Final desde eclipse

* Checkout del proyecto cg_final desde eclipse o desde la consola con:

svn checkout https://cg2010.googlecode.com/svn/trunk/cg_final cg_final

* Desde la consola en el raiz del proyecto correr:

sh lib/mvn-lib-install.sh

* Correr despues la siguiente linea:

mvn clean install exec:java

* Si se quiere usar desde eclipse, correr:

mvn clean install eclipse:eclipse

* Si se quiere correr desde eclipse, ir a Run->External Tools->External Tools Configuration... y crear una nueva configuracion
* En location path, poner: 

/usr/bin/mvn (o donde este el ejecutable de maven)

* En working directory poner:

${workspace_loc:/cg_final} (con cg_final como nombre del proyecto)

* En arguments poner:

exec:java

* Poner Apply y Run.

------------------------------------------------------------------

**Si se quiere tener el source de JME:

svn co -r HEAD http://jmonkeyengine.googlecode.com/svn/trunk jme
